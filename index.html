Non-Contact Sleep Monitor

A full-stack embedded system that monitors sleep vitals in real time without physical contact. An ESP32-S3 reads heart rate and breathing rate from a C1001 mmWave radar over UART and room temperature from a DS18B20 over 1-Wire. Data is averaged over 10-second windows, packed into JSON, and streamed over BLE using a custom NimBLE GATT notify characteristic. A C# server receives the BLE stream, reassembles MTU-chunked packets, and writes sessions and readings to a hosted PostgreSQL database. A live React dashboard displays current vitals and historical session charts.

Fire Alarm — Temperature Sensor Alarm System

A temperature monitoring and alarm system built on a fully custom PCB designed in KiCad. The DS18B20 digital sensor reads ambient temperature every second over 1-Wire and displays it on a TM1650-based 4-digit 7-segment display. A second display shows the configurable alarm threshold at all times. When temperature exceeds the threshold, an alarm LED activates and must be manually acknowledged via an onboard tactile button. The board is USB-C powered with an AMS1117-3.3 LDO and includes an SWD header for in-circuit programming.
